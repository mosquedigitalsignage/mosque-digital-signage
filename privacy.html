Privacy Policy

Last updated: February 22, 2026

Mosque Digital Signage ("we", "our", "the app") is a digital signage platform for mosques. This policy explains how we handle information when you use our Android TV app and web application.

Information We Collect

Mosque administrators sign in with Google to configure their mosque's display settings. We store:

  • Google account email address (for admin authentication)
  • Mosque configuration data (name, location zip code, display preferences)
  • Announcements and custom text entered by the admin

Display viewers and Android TV app users — we do not collect any personal information. The display app loads mosque data from our database and prayer times from a public API. No user accounts, tracking, or analytics are used on the display side.

How We Use Information

  • Admin email is used solely to authenticate and authorize access to mosque settings
  • Zip code is used to fetch accurate prayer times from the Aladhan API
  • Google Drive folder IDs are used to load slideshow images configured by the admin

Data Storage

Mosque configuration data is stored in Google Firebase Firestore. Admin authentication is handled by Firebase Authentication. All data is transmitted over HTTPS.

Third-Party Services

  • Firebase (Google) — authentication and data storage
  • Google Drive API — loading images shared by the admin
  • Aladhan API — prayer time calculations based on zip code

These services have their own privacy policies. We do not sell or share data with any other third parties.

Local Storage

The app caches Google Drive image data in the device's local storage for performance. This cache is automatically refreshed every 24 hours. No personal data is stored locally.

Children's Privacy

Our app does not knowingly collect personal information from children under 13.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated date.

Contact

If you have questions about this privacy policy, contact us at mosquedigitalsignage@gmail.com.
